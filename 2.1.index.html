2.1 Grunderna i CSS 3

Externt länkad CSS-fil och dess fördelar

En externt länkad CSS-fil ger möjligheten att helt separera din CSS från din HTML i olika filer.

Fördelar:

  • CSS:en kan sparas av webbläsaren (cachas) så att den bara behövs laddas in en gång, vilket gör att sidan kan laddas in snabbare nästa gång den besöks.
  • Förbättrar läsbarheten av CSS och HTML filerna.
  • Förenklar återanvändandet av CSS i flera olika HTML-dokument.

Arv

Cascade

Selektorer (tagg-, attribut-, klass-, id- och barn-selektorer)

Enheter (absoluta: in, cm, mm, pt, pc, px och relativa: em, ex, %)
