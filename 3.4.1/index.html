3.4.1 Effekter och animationer

Visar och gömmer ett HTML-element (show, hide och toggle)

HTML

<button class="js-hide">hide()</button>
<button class="js-show">show()</button>
<button class="js-toggle">toggle()</button>
<img class="js-image-a" src="../images/exampleImage.png" alt="Rocket">

JavaScript

$(".js-hide").click((event) => $(".js-image-a").hide() )
$(".js-show").click((event) => $(".js-image-a").show() )
$(".js-toggle").click((event) => $(".js-image-a").toggle() )

Resultat

Fadar in ouch ut ett HTML-element (fadeIn, fadeOut, fadeToggle och fadeTo)

HTML

<button class="js-hide">fadeOut()</button>
<button class="js-show">fadeIn()</button>
<button class="js-toggle">fadeToggle()</button>
<img class="js-image-b" src="../images/exampleImage.png" alt="Rocket">

JavaScript

$(".js-fadeOut").click((event) => $(".js-image-b").fadeOut() )
$(".js-fadeIn").click((event) => $(".js-image-b").fadeIn() )
$(".js-fadeToggle").click((event) => $(".js-image-b").fadeToggle() )

Resultat

Gör även ett program som skapar följande animation:

  • Förändrar ett HTML-elements stilsättning (animate)
  • Easing (linear eller swing) till animationen och testa med olika hastigheter
  • Att en händelse sker efter animationen är klar (callback funktion)

HTML

<button class="js-animate">animate()</button>
<img class="js-image-c" src="../images/exampleImage.png" alt="Rocket">

JavaScript

$(".js-animate").click((event) => $(".js-image-c").animate({
    left: [ "+=200", "swing" ],
    opacity: [ 0.5, "linear" ]
  }, 400, () => { alert("Animation done!") })
);
