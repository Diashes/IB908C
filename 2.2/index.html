2.2 Text och listor

Fontfamiljer

Serif: Fonten har små linjer på bokstavskanterna.

Sans-Serif: Har inga linjer på bokstavskanterna.

Monospace: Alla monospace-bokstäver har samma breli.

Enheter

Extra extra liten text.

p { font-size: xx-small; }

Extra liten text.

p { font-size: x-small; }

Extra extra stor text.

%: Textens storlek visad i procent.

Textens storlek visad i procent.

Textens storlek visad em.

Färg

Fördefinierad färg: Definierar hur tjocka bokstäverna ska vara.

Fördefinierad färg.

p { color: burlywood; }
Egen färg: Du kan även definiera egna färger genom att skriva en hexdecimalt format, RGB-format eller RGBA-format.

Hexadecimalt format färg.

p { color: #bada55; }

RGB-format färg.

p { color: rgb(100, 150, 80); }

RGBA-format färg.

p { color: rgb(100, 150, 80, 0.2); }

Teckensnitt

font-family: Anger vilken teckensnittsfamilj som används.

Anger vilken teckensnittsfamilj som används.

p { font-family: "Comic Sans MS", cursive, sans-serif; }
font-stretch: Med font-stretch egenskapen kan du göra texten bredare eller smalare.

Med font-stretch egenskapen kan du göra texten bredare eller smalare.

p { font-stretch: expanded; }
font-size: Ändrar storleken på texten.

Ändrar storleken på texten.

p { font-size: 3em; }
font-size-adjust: Webbläsaren justerar textstorleken så att den är samma oavsett vilken font som används.

Webbläsaren justerar textstorleken så att den är samma oavsett vilken font som används.

p { font-size-adjust: 0.58; }
font-style: Stylar texten på diverse olika sätt.

Stylar texten på diverse olika sätt.

p { font-style: italic; }
font-variant: Konverterar lowercase till små uppercase-bokstäver.

Konverterar lowercase till små uppercase-bokstäver.

p { font-variant: small-caps; }
font-weight: Definierar hur tjocka bokstäverna ska vara.

Definierar hur tjocka bokstäverna ska vara.

p { font-weight: bold; }
font: Specifiserar font-style, font-variant, font-weight, font-size/line-height och font-family i samma egenskap.

Specifiserar font-style, font-variant, font-weight, font-size/line-height och font-family i samma egenskap.

p { font: italic bold 16px/25px monospace; }

letter-spacing: Sätter avståndet mellan bokstäverna.

Sätter avståndet mellan bokstäverna.

p { letter-spacing: 2px; }
word-spacing: Sätter avståndet mellan ord.

Sätter avståndet mellan ord.

p { word-spacing: 10px; }
line-height: Sätter avståndet till nästa textrad.

Sätter avståndet till nästa textrad.

p { line-height: 10px; }
text-align: Sätter textens justering.

Sätter avståndet mellan bokstäverna.

p { text-align: center; }
text-decoration: Sätter dekoration på texten.

Sätter dekoration på texten.

p { text-decoration: underline; }
text-shadow: Lägger till en skugga bakom texten.

Lägger till en skugga bakom texten.

p { text-shadow: 2px 2px #B4D455; }
text-indent: Indenterar den första raden av texten.

Indenterar den första raden av texten.

p { text-indent: 50px; }
text-transform: Transformerar texten mellan stora och små bokstäver.

Transformerar texten mellan stora och små bokstäver.

p { text-transform: uppercase; }
white-space: Specifiserar om texten ska brytas till nästa rad eller inte.

Specifiserar om texten ska brytas till nästa rad eller inte.

p { white-space: nowrap; }
