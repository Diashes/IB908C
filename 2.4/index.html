2.4 Bilder och bakgrunder

background

bakground kombinerar egenskaperna background-color, background-image, background-repeat, background-attachment och background-position i en och samma egenskap.

background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;
Visa exempel 
div {
    background: #bada55 url("image.png") no-repeat right top;
}
div

background-attachment

Bestämmer om bakgrunden följer med när man skrollar på hemsidan.

background-attachment: scroll|fixed|local|initial|inherit;
Visa exempel 
div {
    background-image: url('image.jpg')
    background-attachment: fixed;
}
div

background-color

Specifiserar en bakgrundsfärg för elementet.

background-color: color|transparent|initial|inherit;
Visa exempel 
div {
    background-color: #b0c4de;
}
div

background-image

Specifiserar en bekgrundsbild för elementet.

background-image: url|none|initial|inherit;
Visa exempel 
div {
    background-image: url('image.jpg');
}
div

background-position

Sätter startpunktspositionen för en bakgrundsbild.

background-position: value;
Visa exempel 
div {
    background-repeat: no-repeat;
    background-image: url('image.jpg');
    background-position: center;
}
div

background-repeat

Definierar om bakgrundsbilden ska upprepas eller inte.

background-repeat: repeat|repeat-x|repeat-y|no-repeat|initial|inherit;
Visa exempel 
div {
    background-repeat: no-repeat;
    background-image: url('image.jpg');
}
div

Bild och text

Om man vill ha bilder i en text så kan det vara användbart att använda sig av egenskapen 'float'. Float gör det möjligt att positionera en bild åt höger eller vänster i texten utan att texten avbryts, den flödar istället runt bilden.

Visa exempel 
img {
float: right;
    margin: 10px;
}
.float h2 {
    clear:both;
}
