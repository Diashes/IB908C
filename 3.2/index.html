3.2 DOM-objekt och jQuery

Ändra en egenskap för ett HTML-elementet

Exempel

Ändra egenskapen background från gul till röd.

HTML

<button id="changePropertyExampleButton">Klicka för att ändra färg</button>
<div id="example1"></div>

CSS

#changePropertyExampleDiv {
  height: 20px;
  width: 20px;
  background: yellow;
}

JavaScript

$("#changePropertyExampleButton").onclick(function() {
  $("#changePropertyExampleDiv").css("background", "red");
});

$("#windowEventWidthExampleSpan").text($(window).width());
$("#windowEventHeightExampleSpan").text($(window).height());
$(window).resize(function() {
  $("#windowEventWidthExampleSpan").text($(window).width());
  $("#windowEventHeightExampleSpan").text($(window).height());

Formulärhändelser

Exempel

Skrivfältets bakgrund ändrar färg när det fokuseras.

HTML

<input type="text" id="formEventsExampleInput" />

JavaScript

$("#formEventsExampleInput").focus(function(){
  $(this).css("background-color", "#cccccc");
  $(this).blur(function(){
    $(this).css("background-color", "#ffffff");
  });
});

Tangentbordshändelser

Exempel

Flytta blocket till höger eller vänster med piltangenterna.

HTML

<div id="keyCodeEventsExampleDiv"></div>

CSS

#keyCodeEventsExampleDiv {
  height: 20px;
  width: 20px;
  background: yellow;
  position: absolute;
  left: 8px;
}

JavaScript

$(document).keydown(function(e) {
  if (e.keyCode == 37) $("#keyCodeEventsExampleDiv").animate({left: "-=5"}, 0);
  if (e.keyCode == 39) $("#keyCodeEventsExampleDiv").animate({left: "+=5"}, 0);
});

Resultat

Kombinerade jQuery händelser med hover och toggle

Exempel

Håll muspekaren över den första boxen för att se ett meddelande. Klicka på den andra boxen för att se ett till meddelande.

HTML

<div id="hoverExampleButton">Hover!</div>
<div id="hoverExampleDiv">"Always have a panda. Pandas are good."</div>
<div id="toggleExampleButton">Klicka!</div>
<div id="toggleExampleDiv">"Always remember that you are absolutely unique. Just like everyone else."</div>

CSS

#changeBorderExampleDiv {
  height: 20px;
  width: 20px;
  background: yellow;
}

JavaScript

$("#hoverExampleButton").hover(
  function() { $("#hoverExampleDiv").toggle(50); },
  function(){ $("#hoverExampleDiv").toggle(50); }
);

          
$("#toggleExampleButton").click(function() {

          
  $("#toggleExampleDiv").toggle(50);

          
});
