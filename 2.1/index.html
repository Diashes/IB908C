Externt länkad CSS-fil och dess fördelar

En externt länkad CSS-fil gör det möjligt att hålla HTML- och CSS-kod separerade. En extern fil har också fördelen att den går att länka till flera olika HTML-dokument, vilket möjliggör att samma stil kan appliceras på flera HTML-dokument utan att upprepa CSS-koden.

<link rel="stylesheet" type="text/css" href="2.1_Grunderna_i_CSS_3.css">

Arv

Många av CSS-reglerna som finns går automatiskt i arv till barn-element av ett föräldra-element.

Cascade

CSS är Cascading Style Sheet, vilket gör det möjligt att definiera en CSS-regel flera gånger för samma element. Om reglerna säger emot varandra så löser CSS det genom att granska rangordningen för reglerna och applicerar den som har högst prioritet. Rangordningen brukar kallas "kaskadordning".

Selektorer

Selektorer används för att CSS ska hitta och kunna manipulera HTML-element. Det finns olika slags selektorer, bland annat id-, tagg-, klass- och barn-selektorer.

Tagg / Element-selektorn markerar alla element som är av en viss typ. Det går att markera alla <p> genom att använda taggen som selektor. 

p {
    color: #B4D455;
}

Id-selektorn använder sig av HTML-attributet id för att identifiera ett element. Id-element används oftast för att unikt kunna identiera ett element på en sida, vilket betyder att det oftast inte finns flera element med samma id-namn. Ett id går till exempel att specifisera genom att skriva <div id="container">. 

#container {
    background: #B4D455;
}

Klass-selektorn liknar id-selektorn och gör det möjligt att definiera klasser för element. Till skillnad från id-selektorn så behöver inte klass-selektorn vara unik för ett element utan kan appliceras på många. Det går att definiera en klass för ett element genom att skriva <div class="post">. 

.post {
    background: #B4D455;
}

Barn-selektorn markerar alla element av en viss typ som ligger inuti ett annat element. För att markera alla barn-element inom ett viss element så kan man skriva som nedan. Koden markerar alla <p> som ligger inuti elementet med klassen '.post'. 

.post p {
    color: #B4D455;
}

Enheter

CSS stödjer användandet av olika enheter för längd. Längdenheter används till exempel för att definiera regler som 'margin', 'width', 'font-size', m.m. Vissa regler tillåter även negativa längder. Det finns två typer av längder; relativa och absoluta längder.

Relativa längder

emRelativt till rot-storleken på texten (4em är 4 gånger så stort som rot-storleken).
exRelativt till höjden av texten.
pxRelativt till skärmens upplösning.
%Relativt till höjden och bredden av omgivningen.

Absoluta längder

cmCentimeter.
mmMillimeter.
inInches.
ptPoints.
pcPicas.

Hur ett CSS-dokument är uppbyggt

För att skriva ett CSS-dokument så måste man veta hur CSS-regler byggs upp.

[selektor] {
    [egenskap]: [värde];
}
[selektor] Här skrivs selektorn för det element man vill manipulera. Några exempel på selektorer är .min-selektor (klass-selektor), #min-selektor (id-selektor), div (tagg/element-selektor).
[egenskap] Här definieras egenskapen som du vill manipulera hos elementet.
[värde] Här skriver du vilket värde den egenskapen ska ha, istället för standardvärdet.
