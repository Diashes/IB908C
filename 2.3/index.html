2.3 CSS Boxmodellen

I CSS kan boxmodellen användas när man pratar om design och layout. Boxmodellen omsluter HTML-element som möjliggör användandet av marginaler, borders och paddning runt elementets innehåll.

  • Innehåll: Innehållet i boxen, där text och bilder visas.
  • Paddning: Ökar avståndet mellan innehållet och boxens kant.
  • Border: En ram som går runt paddningen och innehållet.
  • Margin: Ökar avståndet utanför boxen till innehållet runtomkring.

Marginaler

Marginaler gör området runt elementet större till elementen runtomkring.

Visa exempel 
div
{
    margin: 60px;
    background: #B4D455;
}
A div element.

Border

Border-egenskapen definierar en ram som ska synas runt elementet.

Visa exempel 
div
{
    border: 1px solid black;
}
A div element.

Border color

Egenskapen border-color används för att sätta färgen på en ram. Färgen kan sättas genom:

  • name: specifisera namnet på färgen, såsom "yellow"
  • RGB: specifisera ett RGB-värde, såsom "rgb(255,255,0)"
  • Hex: specifisera ett hex-värde, såsom "#fff000"
  • transparent: sätt färgen till genomskinlig

Egenskapen border-color fungerar inte enskilt, utan man måste först sätta värdet på egenskapen border-style.

Visa exempel 
div
{
    border-style: solid; 
    border-color: green; 
}
A div element.

Border style

Border-style-egenskapen specifiserar vilken slags ram som ska visas. Värden som egenskapen border-style kan ha är:

  • none: ingen ram visas
  • dashed: definierar en streckad ram
  • solid: definierar en solid ram
  • double: definierar en dubbel ram
  • groove: definierar en ram med 3D-effekt
  • m.m...

Om inte border-style-egenskapen sätts så kommer ingen ram att visas.

Visa exempel 
div
{
    border-style: dashed; 
}
A div element.

Border width

Border-width används för att sätta tjockleken på ramen. Tjockleken kan definieras genom pixlar eller någon av de fördefinierade värdena; thin, medium, thick. Observera att innan border-width kan användas så måste border-style sättas.

Visa exempel 
div
{
    border-style: solid;
    border-width: 22px; 
}
A div element.

Paddning

Paddning definierar mellanrummet mellan elementets innehåll och elementets kant.

Visa exempel 
div
{
    padding: 60px;
    background: #B4D455;
}
A div element.

Outlines

Outline används för att visa en linje utanför elementets kanter. Outline liknar border väldigt mycket men den huvudsakliga skillnaden är att outline inte räknas in som en del av elementets storlek. Det medför att ramen du kan "flyta ut" över andra element.

Visa exempel 
div {
    outline: red solid 15px;
}
A div element.

Synlighet

HTML-element har olika standardvärden för synlighet beroende på vilket element det är. De flesta element har värden som block och inline. Synligheten av ett element kan definieras av flera olika CSS egenskaper. Till exempel..

Visibility:

  • visible: standardvärdet, elementet är synligt
  • hidden: gömmer elementet men tar fortfarande upp samma plats

Display: specifiserar hur ett element ska visas:

  • block: visar elementet på nästa rad
  • inline: visar elementet på samma rad
  • none: gömmer elementet helt

Andra:

  • clip: specifiserar en rektangel som klipper ett element med absolut position till en viss storlek
  • overflow: specifiserar vad som händer om innehållet är större än elementets storlek
Visa exempel 
.div1 
{
    display: none;
}
.div2
{
    visibility: hidden;
}
.img3
{
    position:absolute;
    clip: rect(0px,60px,200px,0px);
}
.div4
{
    border: 1px solid gray;
    width: 50px;
    height: 50px;
    overflow: scroll;
}

Design

Designen kan även ändras med hjälp av dessa egenskaper:
  • opacity: sätter opaciteten för det valda elementet.
  • rgba: ett sätt att bestämma en färg och opaciteten på färgen.
  • border-radius: rundar hörnen på det valda elementet.
  • background-image: denna egenskap tillåter användandet av flera bakgrundsbilder, som läggs på varandra där den första bilden hamnar längst upp.
  • box-shadow: lägger till en skugga bakom det valda elementet.
  • linear-gradient, radial-gradient: gör det möjligt att skapa en mjuk övergång av färger, linjärt eller diagonalt.
Visa opacity exempel 
.div 
{
    opacity: 0.5;
    width: 250px;
    height: 250px;
    background-color: red;
}
Ett div element.
Visa background-image exempel 
.div 
{
    width: 250px;
    height: 250px;
    background-repeat: no-repeat;
    background-position: right top, left bottom, left top;
    background-image: url('../images/firstImage.png'), url('../images/secondImage.png'), url('../images/thirdImage.png');
}
Ett div element.
Visa rgba exempel 
.div 
{
    width: 250px;
    height: 250px;
    background-color: rgba(0, 0, 30, 0.3);
}
Ett div element.
Visa box-shadow exempel 
.div 
{
    width: 250px;
    height: 250px;
    box-shadow: 10px 10px 5px #888888;
}
Ett div element.
Visa border-radius exempel 
.div 
{
    width: 250px;
    height: 250px;
    background-color: red;
    border-radius: 10px;
}
Ett div element.
Visa linear-gradient exempel 
.div
{
    width: 250px;
    height: 250px;
    background: linear-gradient(red, rgba(0,0,255,0.3));
}
Ett till div element.
