Uppgift 1.1 Grunderna i HTML5

Doctype

Ett html-dokument börjar alltid med raden <!DOCTYPE html> som berättar för webbläsaren att dokumentet kommer att innehålla HTML5 kod.

Teckenkod

Teckenkod är ett sätt för datorn att förstå hur olika slags tecken ska sparas och sedan kunna läsas för att kunna visas korrekt. Den vanligaste teckenkodningen idag kallas UTF-8 och innehåller stöd för alla tecken man någonsin skulle behöva.

Språk

Du kan specifisera språket genom att lägga till lang="en" i en tagg, vanligast är html-taggen. Det hjälper t.ex. sökmotorer och skärmläsare att identifiera rätt språk på innehållet.

Validering

För att validera att du skrivit korrekt html så finns det html-valideringsverktyg som kan berätta om det finns fel i koden. Du kan t.ex. använda http://validator.w3.org.

Hur ett HTML-dokument är uppbyggt

Ett HTML-dokument är uppbyggt med något som kallas taggar. Taggar används för att skapa olika slags information om HTML-dokumentet. Men innan man kan börja använda taggar så måste man inkludera <!DOCTYPE html> för att berätta för webbläsaren att man vill använda HTML5. En tagg består ofta av en start-tagg och en slut-tagg och mellan taggarna så inkluderar man sin data, t.ex. <title>Min titel</title>. Det enklaste HTML-dokumentet som går att skapa: 

    <!DOCTYPE html>
    <title>Ett HTML-dokument</title>
    <p>En paragraf.</p>

Men det är rekommenderat att man även använder taggarna <head> för meta-information och <body> för synligt innehåll för HTML-dokumentet: 

    <!DOCTYPE html>
    <html>
    <head>
      <title>A Sample Webpage</title>
    </head>
    <body>
    <p>En paragraf.</p>
    </body>
    </html>

Struktur

<body>

Defines the body of the document and it's content.

    <html>
      <body>
        <!-- innehåll för body -->
      </body>
    </html>

<head>

Is a container for metadata and is placed beside the <body> tag. Metadata is not displayed in the document. 

    <html>
      <head>
        <title>Min Titel</title>
      </head>
    </html>

<html>

This tag tells the browser that this document is a HTML document. 

    <html>
      <!-- innehåll för html -->
    </html>

<title>

Defines the title of the document.

    <html>
      <head>
        <title>
            <!-- innehåll för title -->
        </title>
      </head>
    </html>

<meta name="description" content="...">

generator

<meta name="generator" content="...">

keywords

Definiera nyckelord för sökmotorer.
